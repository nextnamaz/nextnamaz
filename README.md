# NextNamaz

Prayer time display platform for mosques. Set up once, display everywhere.

## Stack

Next.js 16 · Supabase · Stripe · Tailwind CSS 4 · Vercel

## Features

- Real-time prayer time displays with customizable themes
- Multi-screen support per mosque
- Google OAuth + email/password authentication
- Stripe billing with free trial
- PWA with offline support

## Getting Started

```bash
npm install
cp .env.example .env.local  # add your keys
npm run dev
```
